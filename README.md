# LyricStation

A simple web app to create .lrc timed with lyrics, integrated with a display using [AMLL](https://github.com/Steve-xmh/applemusic-like-lyrics) library.

The web application is accessible at https://lyricstation.xyz

## Development Setup
using Node.js (version 20 or later).

1. Dependencies
   ```bash
   npm install
   cd app && npm install
   ```
2. Test Server
   ```bash
   npm run dev
   ```
